# SEO Optimization Guide

Comprehensive SEO strategies covering technical implementation, on-page optimization, and Core Web Vitals for modern web applications.

## What This Skill Does

This skill provides expert guidance on search engine optimization, helping you:

- **Implement Technical SEO**: Meta tags, structured data, sitemaps, robots.txt
- **Optimize Core Web Vitals**: LCP, INP, CLS improvements for better rankings
- **Improve On-Page SEO**: URL structure, headings, internal linking
- **Mobile-First Design**: Responsive, fast, touch-friendly implementations
- **Framework Integration**: Next.js, React, and modern framework patterns
- **Monitor Performance**: Tools and metrics for ongoing optimization

## When to Use This Skill

Use this skill when you need to:

- Set up SEO for a new website or application
- Improve search rankings for existing content
- Optimize Core Web Vitals scores
- Implement structured data (Schema.org)
- Fix technical SEO issues
- Create SEO-friendly meta tags and OG images
- Build XML sitemaps and robots.txt
- Optimize page speed for SEO
- Implement mobile-first SEO

   - Robots.txt configuration

   - URL structure best practices
   - Heading hierarchy (H1-H6)

   - Keyword research and implementation

3. **Core Web Vitals**
   - Largest Contentful Paint (LCP) optimization

4. **Mobile-First SEO**

   - Google Search Console setup

   - Regular auditing workflows
   - Performance tracking
